# 🛒 Amazify - E-Commerce Shopping Cart

A lightweight, responsive e-commerce product listing and shopping cart interface. Built with vanilla JavaScript, HTML, and CSS.

## ✨ Features

- **Product Listing** - Display products with images, ratings, and prices
- **Shopping Cart** - Add/remove items, update quantities
- **Local Storage** - Cart persists between browser sessions
- **Order Summary** - Real-time calculation of totals, taxes
- **Responsive Design** - Works on desktop and mobile devices
- **Delete Animation** - Smooth removal of cart items
- **Notification System** - User feedback for cart actions

## 📋 Pages

- **Products Page** - Browse and add items to cart
- **Checkout Page** - Review cart, update quantities, place orders
- **Orders Page** - Track order status (Preparing → Shipped → Delivered)

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Local Storage API
- Git & GitHub
